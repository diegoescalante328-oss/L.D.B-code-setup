# 011.000 What 011_env Is

`011_env/` holds environment setup notes and runbook files for preparing the local Python runtime used by Screen Analyzer V1.

This folder is documentation-only and does not contain secrets.

## Purpose
- Standardize virtual environment setup for local development/runtime.
- Provide a stable place for environment bootstrap steps.
- Keep setup guidance close to the app runtime root.

## Files
- `011.001_virtual_environment_setup.md`: step-by-step virtual environment setup and validation.
