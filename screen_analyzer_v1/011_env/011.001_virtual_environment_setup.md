# 011.001 Virtual Environment Setup

Use this guide to create and validate the Python virtual environment for Screen Analyzer V1.

## Runtime root
Run all commands from:

```bash
cd /workspace/L.D.B-code-setup/screen_analyzer_v1
```

## Linux/macOS

### 1) Create virtual environment
```bash
python -m venv .venv
```

### 2) Activate virtual environment
```bash
source .venv/bin/activate
```

### 3) Install dependencies
```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### 4) Set required API key
```bash
export OPENAI_API_KEY="your_key_here"
```

### 5) Verify Python and key are visible in this shell
```bash
python --version
python -c "import os; print('OPENAI_API_KEY set:', bool(os.getenv('OPENAI_API_KEY')))"
```

## Windows PowerShell

### 1) Create virtual environment
```powershell
python -m venv .venv
```

### 2) Activate virtual environment
```powershell
.\.venv\Scripts\Activate.ps1
```

### 3) Install dependencies
```powershell
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### 4) Set required API key
```powershell
$env:OPENAI_API_KEY="your_key_here"
```

### 5) Verify Python and key are visible in this shell
```powershell
python --version
python -c "import os; print('OPENAI_API_KEY set:', bool(os.getenv('OPENAI_API_KEY')))"
```
