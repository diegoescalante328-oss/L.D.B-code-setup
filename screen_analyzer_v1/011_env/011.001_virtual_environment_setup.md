# 011.001 Virtual Environment Setup (Windows PowerShell)

Use this guide to create and validate the Python virtual environment for Screen Analyzer V1 on Windows.

## Runtime root
Run all commands from:

```powershell
Set-Location .\screen_analyzer_v1
```

## 1) Create virtual environment
```powershell
python -m venv .venv
```

## 2) Activate virtual environment
```powershell
.\.venv\Scripts\Activate.ps1
```

## 3) Install dependencies
```powershell
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## 4) Set required API key
```powershell
$env:OPENAI_API_KEY = "your_key_here"
```

## 5) Verify Python and key are visible in this shell
```powershell
python --version
python -c "import os; print('OPENAI_API_KEY set:', bool(os.getenv('OPENAI_API_KEY')))"
```

## Next steps after environment setup

1. Smoke-test camera:
```powershell
python .\005_scripts\005.002_smoke_capture.py --source 0 --output-dir .\008_outputs\smoke
```

2. Single-frame API check:
```powershell
python .\005_scripts\005.001_single_frame_analysis.py --image .\008_outputs\smoke\<frame>.jpg
```

3. Run full app:
```powershell
python .\001_app\002_app_entrypoint.py
```

## Output locations
- Snapshots: `008_outputs/snapshots/`
- Logs: `010_logs/runtime.jsonl`

## Troubleshooting
If activation fails due to execution policy restrictions:

```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```
