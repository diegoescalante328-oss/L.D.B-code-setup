# 007.004 Setup and Run Guide (Windows PowerShell)

This guide describes the V1 setup and execution flow using **Windows PowerShell** as the primary shell.

## Runtime root
Use `screen_analyzer_v1/` as the canonical runtime root.

```powershell
Set-Location .\screen_analyzer_v1
```

## 1) Create venv and install dependencies

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt
```

## 2) Set API key

```powershell
$env:OPENAI_API_KEY = "your_key_here"
```

## 3) Review runtime config
Open and verify values in `002_config/002.002_runtime_settings.yaml`:

- `camera.source`
- `capture.interval_seconds: 3`
- `capture.max_in_flight_requests: 1`
- `capture.latest_frame_wins: true`

## 4) Smoke-test camera

```powershell
python .\005_scripts\005.002_smoke_capture.py --source 0 --output-dir .\008_outputs\smoke
```

- Press `s` to save a frame.
- Press `q` to quit.

## 5) Single-frame API check

```powershell
python .\005_scripts\005.001_single_frame_analysis.py --image .\008_outputs\smoke\<frame>.jpg
```

## 6) Run full desktop app

```powershell
python .\001_app\002_app_entrypoint.py
```

## 7) Monitor outputs

- Snapshots: `008_outputs/snapshots/`
- Logs: `010_logs/runtime.jsonl`

## Troubleshooting: execution policy blocks activation

If PowerShell prevents script execution, run:

```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```
